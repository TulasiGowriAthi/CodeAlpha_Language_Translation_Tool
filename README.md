### 🌐 AI Language Translation Tool with Text-to-Speech
A production-grade, interactive web application built with Python and Streamlit that provides real-time language translation. The tool features intelligent multi-language processing powered by the Google Translate engine (deep-translator) and an integrated Text-to-Speech (TTS) engine (gTTS) to deliver localized audio pronunciations instantly.
Developed as part of the CodeAlpha Artificial Intelligence Internship.

### 🎯 Key Features
Intuitive Web UI: A clean, centered modern layout built entirely in Python using Streamlit.
Auto-Language Detection: Automatically identifies the incoming source language, requiring no manual configuration from the user.
Expansive Language Support: Seamlessly translates between English, Spanish, French, German, Hindi, Mandarin, Arabic, Japanese, Russian, and Telugu.
Integrated Text-to-Speech (TTS): Generates high-fidelity localized audio clips of the translated text on-the-fly.
Robust Exception Handling: Built-in network timeout and empty input validation to ensure a seamless, crash-free user experience.

### 🚀 Tech Stack & Core Libraries
Frontend / Framework: Streamlit (v1.57.0+)
Translation Engine: Deep-Translator (via Google Translate API integration)
Audio Processing Engine: gTTS (Google Text-to-Speech)
Backend Environment: Python 3.9+
### 🛠️ Installation & Local Setup
1. Download the Project: Download the zip file of this repository and extract it, or open this project folder directly in VS Code.

2. Install Dependencies: Open your VS Code terminal and run:

   pip install -r requirements.txt
3. Launch the Application: Open your VS Code terminal and run:

       streamlit run app.py
